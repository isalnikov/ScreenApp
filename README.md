# ScreenApp - Система управления цифровыми витринами

## Описание проекта

Программное обеспечение для показа текстового, графического и видео контента на витринах (экранах) по всей стране. Система поддерживает управление сетью из ~5000 экранов-телевизоров.

## Технологии

- **Язык программирования**: Java 21
- **Фреймворк**: Spring Boot 3.0
- **База данных**: PostgreSQL / MySQL (настраивается)
- **Frontend**: HTML5 + JavaScript
- **API**: RESTful Public API

## Архитектура системы

### Компоненты

1. **Backend (Spring Boot)**
   - Управление контентом
   - Планирование расписания показа
   - Мониторинг состояния экранов
   - Публичное API

2. **ARM Контента (Web GUI)**
   - Загрузка и управление контентом
   - Создание плейлистов
   - Назначение контента на экраны
   - Статистика и отчеты

3. **Клиентское ПО для экранов**
   - Получение контента
   - Воспроизведение по расписанию
   - Отправка статуса

## Структура базы данных

### Основные сущности

- **Screen** - информация об экране (локация, статус, характеристики)
- **Content** - контент (текст, изображение, видео)
- **Playlist** - плейлист для воспроизведения
- **Schedule** - расписание показа
- **Region** - географические регионы

## Установка и запуск

### Требования

- Java 21+
- Maven 3.8+
- PostgreSQL 14+ или MySQL 8+

### Конфигурация

```bash
# Скопируйте пример конфигурации
cp src/main/resources/application.properties.example src/main/resources/application.properties

# Отредактируйте параметры подключения к БД
```

### Запуск приложения

```bash
# Сборка проекта
mvn clean package

# Запуск
java -jar target/screenapp-1.0.0.jar
```

### Запуск тестов

```bash
mvn test
```

## API Documentation

Публичное API доступно после запуска по адресу: `http://localhost:8080/api/v1`

### Основные эндпоинты

- `GET /screens` - список экранов
- `POST /content` - загрузка контента
- `GET /playlists` - список плейлистов
- `POST /schedules` - создание расписания
